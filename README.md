## Technical report

Clone the repository via
```bash
git clone https://github.com/glimix/qep-paper.git
```

Enter into the `tr` folder and compile the pdf:
```bash
cd qep-paper/tr
xelatex main.tex
open main.pdf
```

As you can see, `tr/main.tex` is the main file for compiling the whole
technical report.
